SEO Exercise 1: Using the Windows 7 IIS SEO toolkit.


  1. We need to work with a Windows 7 OS. Since we're probably already working with Windows 10 then we have to create a W7 virtual machine. If you already have Hyper-V installed on your computer then go to this Google drive and download the W7 ISO file. Use it to create a W7 Professional VM. As soon as it's done, install IIS on the VM (from Control Panel -> Programs -> Windows components on/off). Download the IIS SEO toolkit from right here and install it.
  2. Go to IIS (Administrative Tools) and scroll down until you find the SEO toolkit. Use this tool to analyze this website: http://www.lapdonline.org/. It may take a while before you get the analysis. Go thru it. It's very instructive.
